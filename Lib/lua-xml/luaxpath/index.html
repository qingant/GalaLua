Overview

Based on LuaExpat, this module is an XPath implementation in Lua, similar to TinyXPath.

LuaXPath is free software and uses the same license as Lua 5.1.

Current version

Current version is 1.0 beta. It was developed for Lua 5.1 based on LuaExpat.

Download

LuaXPath can be downloaded in source code from its LuaForge page.

What's new

  • [28/Jan/2008] Version 1.0 beta released

Installation

LuaXPath is a Lua library file called xpath.lua that must be copied to your LUA_PATH.

LuaXPath follows the package proposal for Lua 5.1, therefore this package should be "installed". In other words, if you are using Lua 5.0, the files compat-5.1.c and compat-5.1.h must be used in the compilation and the file compat-5.1.lua must be installed in the LUA_PATH. If you are using Lua 5.1, nothing should be done.
